PQC95 English Wordlist Standard

- Total words: 4096
- Bits per word: 12
- Sorting: alphabetical
- Indexing: zero-based
- Index 0 = first line
- Index 4095 = last line
- Binary format: 12-bit unsigned integer
- Seed format: 32 words
- Entropy: 352 bits
- Checksum: 32 bits
- Total seed bits: 384 bits
